## Projetos

### [client-o-preview](https://github.com/meketreve/client-o-preview)


Aplicativo Windows para pre-visualizacao em miniatura de janelas em tempo real usando a API nativa DWM Thumbnails. Arquitetura em camadas (Models/Views/Services), zoom configuravel, hotkeys por dispositivo de entrada, persistencia em JSON e build de arquivo unico via dotnet publish.

> 5 releases publicadas (v0.1 -> v0.5) - 28 commits

---

### [texuguito-seu-bot-amigo](https://github.com/meketreve/texuguito-seu-bot-amigo)


Bot para Twitch com integracao OAuth2 a API oficial, sistema de pontos local, reproducao de audio via comandos no chat, multiplos aliases, autoreconexao e setup automatizado. Empacotado como executavel standalone com PyInstaller.

---

### [projeto-docker-image-to-ASCII](https://github.com/meketreve/projeto-docker-image-to-ASCII)


Web service conteinerizado para conversao de imagens em arte ASCII. Arquitetura de microsservico com docker-compose, volume mount para desenvolvimento ativo e output via browser ou download .txt.
